# Pawn Repo

**Pawn Repo** is a web-based internal application designed specifically for pawnshop staff. It streamlines daily operations and improves transaction accuracy with a suite of essential tools.

## Features

- 📋 Customer registration and profile management  
- 💰 Pawn transaction recording and tracking  
- 🖼️ Photo uploads for pawned items  
- 🔢 Automatic loan and interest calculation  
- 🔍 Searchable dashboard for quick access to records  
- 🖨️ Receipt generation and printing
